# Citation Standards — 학술 인용 표준

paper-writer / submission-preparer 에이전트의 인용/참고문헌 작성 역량 강화. 국내 측량·공간정보 학술지 투고를 기본 전제로 하되, 해외 저널 대비용 스타일도 함께 다룬다.

## 국내 학술지 국문 참고문헌 작성 표준

국내 측량·공간정보 학회(한국측량학회, 한국지형공간정보학회, 대한원격탐사학회 등)는 대체로 아래와 유사한 국문 참고문헌 양식을 사용한다 (**학회별 세부 양식은 반드시 투고 규정 원문으로 재확인**).

**본문 인용:**
```
- 1저자: (홍길동, 2024)
- 2저자: (홍길동·김철수, 2024)
- 3저자 이상: (홍길동 등, 2024)
- 직접 인용: (홍길동, 2024, p. 42)
- 복수 인용: (홍길동, 2024; 김철수, 2023)
```

**국문 참고문헌 (학술지 논문):**
```
홍길동, 김철수 (2024), 스마트폰 영상 기반 3D Gaussian Splatting의
측량 정확도 검증에 관한 연구, 한국측량학회지, 42(3), pp. 123-135.
```

**영문 참고문헌 (해외 저널/학위논문 인용 시):**
```
Kerbl, B., Kopanas, G., Leimkühler, T., and Drettakis, G. (2023),
3D Gaussian Splatting for Real-Time Radiance Field Rendering,
ACM Transactions on Graphics, 42(4), pp. 1-14.
```

**학위논문:**
```
홍길동 (2024), 3D Gaussian Splatting을 이용한 현장 측량 정확도 검증,
석사학위논문, ○○대학교.
```

**웹/기술문서:**
```
국토지리정보원 (2024), 수치지형도 작성 작업규정, 국토지리정보원.
```

## 참고: 국제 인용 스타일 (해외 저널 투고 대비)

국내 학술지 투고가 기본값이나, 해외 저널(ISPRS Journal, Remote Sensing, Sensors 등)로 전환하거나 영문 학술대회에 투고할 경우를 대비해 아래 스타일도 참조한다.

### APA 7th Edition
```
본문: (Kim, 2024) / (Kim & Lee, 2024) / (Kim et al., 2024)
참고문헌:
  Kim, J. H., & Lee, S. (2024). Title of article.
  Journal Name, 15(2), 123-145. https://doi.org/10.xxxx/xxxxx
```

### IEEE (사진측량/영상처리/컴퓨터비전 계열 학회에서 흔히 사용)
```
본문: [1], [2], [3] / "As shown in [1]..."
참고문헌:
  [1] J. H. Kim and S. Lee, "Title of article,"
      J. Name, vol. 15, no. 2, pp. 123-145, 2024.
```

## 학술지별 인용 스타일 매핑

| 분야/학회 | 대표 학술지 | 인용 스타일 |
|----------|-----------|-----------|
| 측량/공간정보(국내) | 한국측량학회지, 한국지형공간정보학회지 | 학회 자체 양식(국문, APA 유사) |
| 원격탐사(국내) | 대한원격탐사학회지 | 학회 자체 양식 |
| 사진측량/컴퓨터비전(해외) | ISPRS Journal, IEEE 계열 | IEEE 또는 APA |
| 원격탐사(해외) | Remote Sensing, Sensors (MDPI) | APA 유사(저널 자체 규정) |

## 인용 원칙

### 인용해야 하는 경우

| 상황 | 인용 유형 |
|------|----------|
| 타인의 알고리즘/기법 차용 | 방법 인용 (예: 3DGS 원 논문, SfM 도구) |
| 정확도 기준/표준 인용 | 표준 인용 (예: ASPRS 기준, 국토지리정보원 규정) |
| 데이터/통계 인용 | 출처 인용 |
| 정확한 문구 사용 | 직접 인용 (따옴표) |
| 이론적 프레임워크 | 원저 인용 |

### 자기 표절 방지

```
- 학위논문/학술대회 발표 자료를 학술지 논문으로 재구성 시 명시적 언급
- 동일 데이터셋을 복수 논문에 사용 시 이전 발표 언급
```

## 논문 구조 표준 (IMRaD, 국내 학술지 관행)

| 섹션 | 내용 | 국내 학술지 관행 |
|------|------|----------------|
| 국문초록/영문초록 | 목적, 방법, 결과, 결론 | 대부분 국영문 병기 요구 |
| 서론 | 배경, 선행연구, 연구 질문 | |
| 연구방법 | 연구 설계, 대상 현장, 절차, 분석 | "참여자" 대신 "연구 대상 현장 및 데이터" |
| 결과 | 발견사항, 정확도/통계 결과 | |
| 고찰 | 해석, 한계, 시사점 | |

### 정확도/통계 결과 보고 표준

```
좌표 정확도 (점 기반):
  RMSE(수평) = X.X cm, RMSE(수직) = X.X cm

점군 비교 (면 기반):
  C2C 평균거리 = X.X cm, M3C2 평균거리 = X.X cm (신뢰수준 95%)

t-test: t(df) = 값, p = 값, d = 값
  예: t(28) = 2.34, p = .023, d = 0.67

ANOVA: F(df1, df2) = 값, p = 값, η² = 값
  예: F(2, 45) = 4.56, p = .013, η² = .095

상관: r(df) = 값, p = 값
  예: r(28) = .45, p < .001
```

## 투고 체크리스트

### 원고 준비
- [ ] 학회 투고 규정(원고작성요령) 확인
- [ ] 분량/자수 제한 준수
- [ ] 인용 스타일 통일 (국문/영문 혼용 시 일관성)
- [ ] 참고문헌 본문 인용과 일치 확인
- [ ] 표/그림 번호 순서, 국영문 캡션 여부
- [ ] 국문초록/영문초록 분량 확인
- [ ] 주제어(키워드) 국영문 병기

### 커버레터/투고신청서 구조
```
1. 논문 제목 및 저자
2. 논문의 핵심 기여 (실무 적용성 포함)
3. 학술지 적합성 설명
4. 연구윤리 준수 확인 (표절검사 등)
5. 저자 정보 및 연락처
```

## 품질 체크리스트

| 항목 | 기준 |
|------|------|
| 인용 스타일 | 학회 투고 규정 준수 |
| 일관성 | 본문 인용 ↔ 참고문헌 일치 |
| 최신성 | 최근 5년 인용 비중 확보 (3DGS 등 최신 기술 분야는 최근성이 특히 중요) |
| 표준 인용 | 정확도 판정에 사용한 표준/규정을 명확히 인용했는가 |
| 표절 | 표절검사(카피킬러 등) 결과 확인 |
| 정확도/통계 보고 | 단위(cm/mm) 및 보고 형식 일관성 |
