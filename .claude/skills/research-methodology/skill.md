# Research Methodology — 3DGS 측량 정확도 검증 방법론

research-designer / statistical-analyst 에이전트의 현장 실증연구 설계 및 정확도 분석 역량 강화. 스마트폰 영상 기반 3DGS의 측량학적 적용성 검증에 특화된 방법론 가이드.

## 연구 설계 유형

### 정확도 검증 연구의 3가지 설계 축

| 설계 | 목적 | 비교 대상 | 강점 |
|------|------|----------|------|
| 절대정확도 검증 (Absolute Accuracy) | 3DGS 산출물의 절대 좌표 정확도 평가 | GNSS-RTK GCP, 국가기준점 | 실무 측량 성과와 직접 비교 가능 |
| 상대정확도/형상정확도 검증 (Relative/Shape Accuracy) | 3DGS 산출물의 형상·치수 정밀도 평가 | TLS 포인트클라우드, TS 관측망 | 절대좌표 없이도 형상 왜곡 진단 가능 |
| 비교연구 (Benchmark Comparison) | 3DGS를 타 재구성 기법과 비교 | 전통 SfM-MVS, UAV 사진측량 | 3DGS의 상대적 위치(정확도/비용/시간) 규명 |

**본 연구(현장 적용성 검증)에는 절대정확도 검증을 기본 축으로 하고, 상대정확도(형상) 검증을 보조 축으로 병행하는 것을 권장한다.**

## 3DGS 처리 파이프라인 개요 (연구설계·프로토콜 설계 시 참조)

```
스마트폰 영상 촬영
  → 프레임 추출 (fps 샘플링)
  → SfM 초기화 (예: COLMAP — 카메라 포즈/내부표정요소 추정)
  → 3DGS 학습 (Gaussian 초기화 → 최적화 → densification/pruning)
  → 좌표계 정합 (GCP 기반 Helmert 변환 또는 ICP)
  → 정확도 평가 (기준측량 데이터와 비교)
```

각 단계는 오차 전파(error propagation) 가능성이 있으므로, 연구 설계 단계에서 **오차원(source of error)을 단계별로 구분**해 기술하는 것이 방법론적 엄밀성을 높인다 (예: SfM 포즈 추정 오차, 3DGS 학습 수렴 오차, 좌표 정합 오차를 분리하여 논의).

## 기준측량(Ground Truth) 선정 프레임워크

| 기준 | GNSS-RTK GCP | 토탈스테이션(TS) | 지상라이다(TLS) |
|------|-------------|-----------------|-----------------|
| 비교 단위 | 이산점(discrete point) | 이산점(discrete point) | 연속 표면(point cloud) |
| 3DGS와의 비교 방식 | 대응점 좌표차 | 대응점 좌표차 | Cloud-to-Cloud/M3C2 |
| 실외 개활지 적합성 | 높음 | 중간(시준 필요) | 높음 |
| 실내/차폐지역 적합성 | 낮음(위성 수신 불량) | 높음 | 높음 |
| 형상(비평면) 왜곡 진단 | 불가(점 단위) | 제한적 | 가능(면 전체 비교) |

**의사결정 가이드**: 연구 목적이 "좌표 정확도"에 집중되면 GCP/TS, "형상·표면 재현 품질"까지 다루려면 TLS 병행을 권장한다.

## 정확도 평가 표준 (참조 규격)

| 표준/규정 | 적용 분야 | 주요 내용 |
|----------|----------|----------|
| ASPRS Positional Accuracy Standards for Digital Geospatial Data | 사진측량/원격탐사 국제 표준 | RMSE 기반 수평/수직 정확도 등급 |
| ISO 17123 시리즈 | 측량기기 정확도 시험 | GNSS/TS 등 장비 자체 정확도 검증 |
| 국토지리정보원 수치지형도 작성 작업규정 | 국내 지형도/공간정보 구축 | 축척별 허용 정확도 기준 |

연구설계 단계에서 위 표준 중 하나를 **채택 근거와 함께 명시**하면, 통계분석가가 산출한 RMSE/C2C 값을 "충족/미충족"으로 판정할 수 있는 정량적 기준선이 된다.

## 표본(현장/반복 촬영) 설계

전통적 사회과학의 표본크기 공식(검정력 분석)은 이 연구 맥락에서 "촬영 반복 횟수" 또는 "대상 현장 수" 설계로 대체된다.

```
현장 수 설계 근거 예시:
  - 현장 유형의 다양성 확보 (예: 개활지 / 구조물 밀집지 / 실내 등 최소 2~3개 유형)
  - 각 유형별 최소 1개 이상 현장에서 반복 촬영(예: 3회 이상)하여
    기기 재현성(repeatability) 자체도 평가 가능하도록 설계

촬영 반복의 목적:
  - 동일 현장 반복 촬영 → 3DGS 재구성의 재현성(precision) 평가
  - 상이 현장 촬영 → 일반화 가능성(generalizability) 평가
```

## 통계 분석 방법 선택

### 분석 방법 의사결정 트리

```
Q1: 기준측량이 점(point) 기반인가, 면(surface) 기반인가?
├── 점 기반(GCP/TS) → 좌표축별 오차, RMSE, MAE, bias
└── 면 기반(TLS) → C2C, M3C2, Completeness

Q2: 조건(중첩률/조도/현장유형)별 정확도 차이를 비교하는가?
├── 2조건 → t-test / Mann-Whitney U
├── 3조건+ → ANOVA / Kruskal-Wallis
└── 연속형 조건(중첩률 %, 프레임 간격 등) → 회귀분석
```

### 주요 통계/정확도 방법 요약

| 방법 | 목적 | 조건 | 보고 항목 |
|------|------|------|----------|
| RMSE/MAE | 점 대 점 정확도 | 대응점 확보 | 값(cm/mm), 허용기준 대비 판정 |
| C2C/M3C2 | 점군 대 점군 정확도 | 정합 완료된 두 점군 | 평균/표준편차 거리 |
| ICP/Helmert 정합 | 좌표계 정합 | 대응점(3개 이상) 또는 중첩 형상 | 정합 잔차(RMSE) |
| t-test/ANOVA | 조건별 정확도 차이 | 정규성, 등분산 | t 또는 F, df, p, 효과크기 |

### 효과 크기 가이드 (조건별 비교 시)

| 지표 | 작은 | 중간 | 큰 |
|------|------|------|-----|
| Cohen's d | 0.2 | 0.5 | 0.8 |
| η² (ANOVA) | 0.01 | 0.06 | 0.14 |

## 연구 윤리 및 실무적 고려사항

이 연구는 인간 대상 연구가 아니므로 IRB 대상이 아니다. 대신 아래를 확인한다.

| 항목 | 확인 |
|------|------|
| 현장 접근 허가 | 사유지/관리시설 촬영 시 소유자·관리자 동의 |
| 데이터 보안 | 민감시설(보안구역 등) 촬영 제한 여부 |
| 항공촬영 규정 | UAV 병행 시 항공안전법 등 관련 규정(해당 시) |
| 저작권 | 기존 GIS/측량 데이터 활용 시 출처 및 라이선스 확인 |

| 항목 | 기준 |

| 기준측량 선정 | 목적(좌표 vs 형상)에 맞는 방법 채택 + 공인 정확도 명시 |
